# UDO Framework v2.0

This is the immutable core framework for the Universal Dynamic Orchestrator (UDO).

## Quick Start

1. Read **START_HERE.md** first
2. Review **ORCHESTRATOR.md** for full specification
3. Check **COMMANDS.md** for available operations
4. Study **HARD_STOPS.md** for critical rules

## What's Included

- **Core Protocols**: Session management, reasoning standards, evidence requirements
- **Infrastructure**: Bridge for multi-agent coordination, templates for standardization
- **Tools & Utilities**: Reusable components for agents and projects
- **Documentation**: Complete specification and guidelines

## Key Principles

1. **Protocol is the Product** - The framework defines the standards
2. **Immutability** - This folder does not change per-project
3. **Inheritance** - Projects inherit from and reference this framework
4. **Isolation** - Project state is completely separate (in UDO Project folder)
