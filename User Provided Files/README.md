# User Provided Files

This folder is for **external references, research materials, and handoff documents** that you bring into the project from outside sources.

## What Goes Here

- **Research documents** — Industry reports, competitive analysis, reference materials
- **External handoffs** — Handoff documents from other sessions or projects
- **Reference materials** — Context documents, briefs, specifications from stakeholders
- **External communications** — Emails, notes, or documentation from external sources
- **Temporary inputs** — Files you're working with that don't yet belong in the framework

## What Does NOT Go Here

- **Session work products** → Go to `/UDO Project/.project-catalog/`
- **AI-generated outputs** → Go to `/UDO Project/.outputs/`
- **Memory/facts** → Go to `/UDO Project/.memory/`
- **Framework files** → Stay in `/UDO Framework/` (immutable reference)

## How It's Used

When you start a session:
1. Reference materials live here temporarily
2. During work, relevant information flows into `/UDO Project/` folders
3. At session end, old references can be archived or deleted

This keeps the working directories clean while preserving external context.

---

**Location in structure:**
- Main project level (not inside UDO Framework or UDO Project)
- Accessible as reference material by both framework and project contexts
