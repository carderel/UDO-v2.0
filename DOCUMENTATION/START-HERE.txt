KEY POINTS
================================================================================

WHAT IS UDO v2.0?
  Universal Dynamic Orchestrator - a framework for AI session management
  with protocol-driven operation, immutable framework + mutable projects

FRAMEWORK vs PROJECT
  Framework: Immutable core (read-only, shared across projects)
  Project:   Mutable state (project-specific, project-specific)

WHY THIS STRUCTURE?
  Clear separation prevents conflicts and ensures consistency
  Framework updates propagate automatically
  Projects are independent yet standardized

WHAT'S READY TO USE?
  Everything - all directories, templates, and documentation
  Just need to customize with project-specific values
